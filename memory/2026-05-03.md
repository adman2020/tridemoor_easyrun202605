### 匹配算法
1. FinishRun 完成后，若 route_id 为空 → goroutine 异步执行
2. 加载 GPS 采样 → 计算外包框 + 中心点
3. NearbyRoutes 查候选路线（外扩 5km）
4. 对每条路线：空间网格索引（0.0003° 格子）+ 3x3 邻域搜索
5. 最近距离 ≤ 30m 计为匹配
6. 最高重合率 ≥ 80% 自动关联路线 + 更新排行榜
